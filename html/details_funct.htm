TeraVue provides a comprehensive image processing environment for efficient integration and management of Earth Observation data.

See examples from sample applications

 

1. Windows environment.

  • Interactive image processing
  • multispectral remote sensing
  • raster/vector data bridging
  • image rectification
  • supports a large variety of image file formats

User friendly interface requires no prior computer knowledge. Object-oriented Multiple Document Interface (MDI) eases data handling of complex multiple image projects. Drag and drop files from the Windows file manager. Short learning curve thanks to on-line help and a detailed tutorial.

2. Comprehensive set of functions

Multispectral image data management. Interactive contrast and color manipulations. Multispectral combinations. Polygon digitizing. Supervised Bayesian classification. Principal component analysis. Unsupervised classifier. Convolution filtering. Edge enhancement. Morphology operators. Raster to vector contour extraction. Vector map overlay. Geometric rectification with GCPs. Image resampling. Geolocation on World Atlas. Color draping on 3D perspective plots. Atmospheric modelling. Spectral measurements simulation.

3. Extended data formats

  • Direct bridging to 8 bit files in popular formats: BMP, LANDSAT, SPOT, IDRISI, GRASS ... up to a size of 32767x32767
  • Bridging to any BSQ, BIL or BIP file structure without duplication of data.
  • It includes integration of 24 bit color BMP.
  • Bridges to popular formats such as : TIF, JPEG, PCX, GIF, TGA up to 4096x4096 sized files

4. Distributed data management

With its unique method of indirect pointing, TeraVue can manage a large number of images on multiple volumes, CD-ROMs and over a network. As many as 256 layers may be linked to a single image.
